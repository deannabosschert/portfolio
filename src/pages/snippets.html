Snippets

I've been working as a (student) teaching assistent at CMD Amsterdam for 2 years now and in this period of time, I've created a lot of code snippets.

Why? Because I wanted to help and teach students by using small, simple examples. Most examples you'll find on the web are filled with side stuff that's only confusing for beginners.

Even though they're fairly simple, I think that making those snippets has contributed lots towards my view on coding and skills

Normally, I wouldn't use iFrames but for embedding these snippets.. well, it's in terms of UX the better solution.

Of course, I could've just included a link to my codepen but who's gonna click on that..

API usage

I've tested out a few different API's. Some are overly explained in comments; this is done for teaching purposes.
