Tech-related

Because i've been wandering the tech-track of Communication and Multimedia Design, I've worked on lots of tech-related projects. Below are a few of those.

functional programming

Data cleaning exercises. Fetch a dataset of a questionnaire that the students filled in during the Information Design Datavisualization-course, and clean this data using filters based on functional programming principles. Project during the Tech Track of Information Design Project 2020.

photography on-the-go

An interactive multi-device prototype for photographers on the go. Travelling photographers can view, rate and annotate their pictures using their phone. Assignment for the second year's project 'Project Web' of CMD.

plog de dag

Responsive onepage design. Website for promoting the 'Plog de dag'-event in the style of IAMsterdam. Added interaction with subtle changes in transitions and with animations.

spotify lyrics

Lyrics design for Spotify. Title design for the song "I'm OK' by Little Big, based on visual research.

design patterns

Multi-Device Design use case analysis. Research of an existing Multi-Device design, presented in a screenflow.

datavisualization

Storytelling-based series of datavisualizations. Digital prototype, presented in a 'choose your story'-style, based on data from a survey.

photography collection nmvw

Client-side application using Vue 2. Dynamically renders data to views using JavaScript and a front-end framework. Includes reflection on the merits and costs of frameworks together.

zorgcowboys

'Zorgcowboys' team project for KRO-NCRV Pointer. Users can find their own healthcare institution or an institution of interest and can shop with the profit they made in a specific year. Users can buy silly things in the webshop like a private jet or an Aston Martin supercar.
